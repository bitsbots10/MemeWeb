TOML Hosting Guide

Multiple ways to host your token's TOML file

Important Note About TOML Files

TOML files are not required to launch your token on the XRP Ledger. They are optional and only provide additional metadata about your token, such as project details, social links, and contact information. You can create and launch your token without a TOML file and add it later if desired.

Using Your Own Domain (Recommended)

If you already have your own domain, this is the simplest and most professional way to host your TOML file:

  1. Create a folder named .well-known in your website's root directory
  2. Create a file named xrp-ledger.toml inside the .well-known folder
  3. Add your token information to the TOML file
  4. Upload both the folder and file to your web server via FTP or your hosting control panel
  5. Your TOML file should now be accessible at: https://yourdomain.com/.well-known/xrp-ledger.toml
