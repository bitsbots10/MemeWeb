What is Auto Sniper / Bundler Bot?

A powerful Telegram bot that helps you snipe dev wallets, detect new tokens, and manage your XRPL transactions

Telegram Bot Interface

Access powerful XRPL trading features through an intuitive Telegram bot interface. Simple commands let you manage wallets, configure settings, and execute trades.

Token Sniping

Automatically detect and snipe promising new tokens and dev wallets on the XRP Ledger. The Global Monitor feature watches for token creations and executes trades based on your parameters.

Wallet Management

Add existing wallets, generate new ones, view balances, and manage multiple XRPL wallets directly from the bot interface.

AMM Monitoring

Monitor specific issuers for AMM (Automated Market Maker) creation events and automatically execute transactions when new opportunities arise.

How to Use Auto Sniper / Bundler Bot

A step-by-step guide to getting started with the Auto Sniper / Bundler Bot

1

Access the Bot

Search for the Auto Sniper / Bundler Bot on Telegram, start a conversation, and use /menu to access the main menu.

2

Set Up Your Wallet

Add your existing wallet by entering your sEd format seed, or generate a new XRP wallet directly in the bot.

3

Configure Settings

Set your transaction parameters including transaction count, XRP amounts, and slippage tolerance.

4

Choose Your Strategy

Decide between batch token purchases, auto AMM mode for specific issuers, or global monitoring for new token detection.

5

Monitor & Manage

Track your transactions, check wallet balances, and use the bot's transaction logs to monitor your activity.

Key Features

Powerful capabilities of the Auto Sniper / Bundler Bot

Global Monitor & Dev Wallet Sniping

The Global Monitor continuously watches the XRP Ledger for new token creations and dev wallet activity, automatically detecting promising tokens and instantly sniping them based on your configured parameters. Settings include auto-sell options, exclusion filters for specific issuers, and customizable transaction parameters.

Auto AMM Mode

Monitor specific issuers for AMM creation events and automatically execute transactions when an AMM is created. Configure settings such as using all wallets or just your main wallet, setting XRP ranges for transactions, and enabling auto-sell functionality for tokens after purchase.

Batch Token Purchases

Initiate batch token purchases with your configured settings, allowing you to execute multiple transactions according to your predefined parameters. This feature optimizes your trading strategy by enabling efficient execution of your investment plan.

Multi-Wallet Support

Manage multiple XRP wallets simultaneously, allowing you to diversify your trading strategy across different accounts. Add existing wallets using sEd format seeds, generate new secure wallets directly in the bot, view wallet addresses and balances, and remove wallets you no longer need.
